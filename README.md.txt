# 👁️ Face Recognition Surveillance System

A real-time face recognition desktop application designed for surveillance and alerting, built using Python, OpenCV, and Tkinter. Detects known or unknown individuals and sends email alerts with captured image and time stamp.

---

## 🚀 Features

- 📸 Real-time face detection & recognition
- 📬 Sends email alerts on unknown face detection
- 🖥️ GUI built with Tkinter for easy interaction
- 📂 Stores captured faces with time stamp
- 🧠 Easy training of new faces
- 📍 Future: Location tagging and cloud storage

---

## 🛠️ Tech Stack

- Python
- OpenCV
- Tkinter
- smtplib (email alert system)

---

## 🧑‍💻 How to Run

1. Clone the repo:
   ```bash
   git clone https://github.com/jyoti90655/Face-Recognition-Surveillance
   cd Face-Recognition-Surveillance
2. Install dependencies:
   pip install -r requirements.txt
3. Run the main program:
   python main.py

 Use Cases:
 -Police surveillance & missing persons tracking
 -Office/College attendance systems
 -Home security systems
 -Retail surveillance for VIPs or blacklist monitoring
