# Face Recognition Surveillance System 🧠📷

A Python-based real-time face recognition system that detects missing persons via webcam, shows personal info, and sends email + location alerts.

## Features
- Real-time face detection using OpenCV
- Face matching with stored images
- Email alerts with location
- CSV-based person info
- GUI interface

## 🚀 Technologies Used
Python, OpenCV, face_recognition, geocoder, pyautogui, pandas, Tkinter

## 👣 How to Run
1. Clone/download the repo
2. Install requirements:
pip install -r requirements.txt

3. Run this:
python face_gui_app.py
